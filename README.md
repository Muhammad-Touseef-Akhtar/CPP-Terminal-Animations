# Procedural C++ Terminal Animations

A collection of math-driven terminal animations and geometric graphics simulations built in C++. This project handles dynamic screen frames in real time using pure logic loops and native Win32 console position coordinates (`windows.h`).


## Animation Showcase

### 1. Inward Spiral & Box Matrix Loop
* **Mechanics:** Loops an inward-turning step sequence to trace custom dimensions, cleanly deleting its path afterward using trailing space tracking variables.

### 2. Polar Geometric Smiley Face
* **Mechanics:** Converts angles to radians to solve standard circle equations ($x = h + r\cos\theta$), mapping facial layers and smooth lower mouth arcs.

### 3. Procedural Character Face
* **Mechanics:** Extends trigonometric rendering logic to calculate asymmetric dimensions including layered glasses, nose structures, and boundary framing.

### 4. Bouncing Shifting Matrix Container
* **Mechanics:** Tracks multi-variable coordinate offsets to dynamically skew solid triangles while animating an isolated particle down a tracking rail.

### 5. Concentric Radar Pulse Ripples
* **Mechanics:** Monitored via variable scaling metrics to expand concentric circles outward across the screen grid buffer before reversing back into the origin coordinates.


## How to Compile and Run

These simulations utilize native screen handles and must be executed within a Windows environment.

1. **Clone the repository:**
   git clone https://github.com

2. **Compile an animation script:**
   g++ 01_Spiral_Loop.cpp -o SpiralAnimation.exe

3. **Run the application:**
   .\SpiralAnimation.exe
